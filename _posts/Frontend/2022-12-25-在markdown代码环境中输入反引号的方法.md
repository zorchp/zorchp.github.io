# 方法

行间代码当然可以直接输入了, 无压力:

````bash
```bash
这里是反引号
```
````

这时候如果想嵌套表示, 即在代码环境中表示代码环境, 那么就可以这样写:

`````bash
````
```bash
a=$s
```
````
`````



那么在行内公式中呢? 直接反斜杠转义是不行的, 但是可以用两对反引号来表示, 像下面这样:

```bash
``这里是单个反引号`, 你也来试试``
```

得到的结果如下: 

``这里是单个反引号`, 你也来试试``

那么, 如果只想表示单个的反引号呢? 也很简单:

```bash
`` ` ``
```

用空格隔开即可, 表示如下:

`` ` ``



# 总结

用数量多的反引号来嵌套数量少的反引号即可.
