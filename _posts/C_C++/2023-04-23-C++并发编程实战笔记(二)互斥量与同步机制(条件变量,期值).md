## 期值(future)



获取异步任务的返回值

```cpp
#include <future>
#include <iostream>
using namespace std;

int find_ans() { return 42; }


int main(int argc, char *argv[]) {
    // async 和 thread 的创建方式类似, 都是先传入函数签名(可调用对象), 然后传入函数参数
    future<int> ans = std::async(find_ans);
    cout << "ans is " << ans.get() << endl; // ans is 42
    return 0;
}
```
