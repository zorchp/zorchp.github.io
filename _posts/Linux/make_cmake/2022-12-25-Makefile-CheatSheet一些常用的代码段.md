# 基本格式

```makefile
target ... : prerequisites ...
	command
```



| 自动化变量 |                            含义                            |
| :--------: | :--------------------------------------------------------: |
|    `$@`    |                         目标文件名                         |
|    `$<`    |              第一个先决条件(文件)的名称(常用)              |
|    `$^`    |        所有先决条件的名称(去重, 每个规则只出现一次)        |
|    `$+`    | 规则中**所有**先决条件列表(包括出现一次的和重复出现多次的) |
